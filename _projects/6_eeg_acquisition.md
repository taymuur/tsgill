Designed and developed a comprehensive EEG acquisition system from ground up, addressing the need for affordable research-grade neural signal recording equipment. The system features custom amplification circuits, Arduino-based control systems, and real-time signal visualization capabilities.

### Key Achievements

- Successfully amplified EEG signals by **150x with minimal noise**
- Implemented **12-bit ADC** for high-resolution signal digitization
- Conducted live testing with Ambu EEG electrodes
- Created cost-effective alternative to commercial EEG systems (**> 80% cost reduction**)
- Demonstrated technical versatility beyond software development

### Technical Approach

The system features:

- **Custom Amplifier Design** - Differential inputs for noise rejection
- **Arduino Control** - Digital data acquisition and processing
- **Signal Chain** - Optimized for biological signal quality
- **PCB Layout** - Careful design for minimal electromagnetic interference
